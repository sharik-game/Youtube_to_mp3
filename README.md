# Simple converter

This is web converter(*writing by fastapi*) can transform .mp4 file to .mp3 file and Youtube link to .mp3 file.  
It can be useful when you don't need image.  
  
---
## Getting starting
- download this repo
- ```python -m venv venv```
- activate your venv
- ```pip install -r requirements.txt```
- ```uvicorn main:app```  
**And then in browser**
- http://localhost:8000
